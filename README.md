## What This Public Repo Contains

This repository is intentionally public and small. It contains only:

- a static landing page,
- public screenshots,
- public technical-report copies,
- submission links and notes.

It does not contain the private iOS source code, model files, API keys, health data, or user data.

## Why A Separate Public Repo

The main 100 Years repository is private because the app contains unreleased iOS code and health-data related implementation details. GitHub does not provide a reliable "make one file public" mode for a private repository, so this public repository acts as the online demo URL for reviewers.

For Track C Edge AI, the actual demo runs on a real iPhone. This page is the public online entry point for the video, report, repository pointer, and feature support matrix.

## Demo Focus

The demo should foreground large-model execution evidence:

1. Local Gemma 4 E2B/E4B model state and receipt.
2. Gemma 4 tool calling for the Today health brief.
3. Private medical history entering local prompt context and disease-risk scoring.
4. Structured local model output for compact longevity analysis.
5. Local-first report/PDF parsing.
6. Edge/cloud boundary: local, edge-cloud combined, cloud-only, and validation-gated tasks.
